# Workflow: Codex Fix Review Blockers

Use after Claude or Codex review finds blockers.

## Steps

1. List blockers.
2. Confirm each blocker is in scope.
3. Apply smallest fixes.
4. Re-run relevant checks.
5. Update status docs if risk/state changed.
6. Report what was fixed and what remains.

## Rules

- Codex applies fixes.
- Claude does not edit.
- Do not bundle unrelated cleanup.
- Do not commit without human approval.
